004 나이출력

문제
2022년 기준 선생님의 나이 age가 주어질 때.
선생님의 출생 연도를 return 하는
solution 함수를 완성하시오
제한사항
0 < age ≤ 120
나이는 태어난 연도에 1살이며 매년 1월 1일마다 1살씩 증가합니다.

class Solution {
    public int solution(int age) {
        int currentYear = 2022; //현재 연도
        int birthYear = currentYear - age + 1; // 출생 연도 계산
        return birthYear;
    }
}
